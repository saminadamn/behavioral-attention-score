Behavioral Attention Score (BAS) is a deterministic multi-agent pipeline for adaptive learning research: a synthetic dataset generator, an explainable attention score, a decomposed reward signal, a rule-based intervention planner, a LangGraph orchestration layer, and an evaluation framework (benchmarks, ablations, sensitivity sweeps). Evidence Aggregation Engine & Logic

The Behavioural Attention Score aggregates weighted normalized evidence. When a feature is missing, the remaining weights are renormalized rather than the feature silently counting as zero.

Formula (thesis §3.7 · Module 9)
Sraw = clip( Σi wi·ei / Σi wi , l, u )
Rt = Rperformance + Rbehaviour − Rcost  (Module 10, exact identity)
wi (Weights):

Configured coefficients per behavioural marker (correctness, latency, coherence, fatigue…).

ei (Evidence):

Normalized values in [0, 1], with positive/negative polarity mapped before aggregation.

l, u (Bounds):

Clip limits constraining the raw score to [0.0, 1.0].
